Nine10 Photo Gallery

Nine10 Photo Gallery Plugin

What is it?

I built a simple plugin that took photos from the first native WordPress photo gallery on a page and placed them in a new layout that ordered them in a list form on the left hand side with a large viewer on the right hand side.

Skills Used/ Learned

This was my first big project with PHP and it went quite well. The php just identifies if their is a gallery and then takes to image urls and generates new html markup. Then I used some simple jQuery to handle the phots switching and highlighting.

What's Next

Now I am working on implementing something very similar on the Client Portal private galleries so clients can view their personal photos more easily.

Nine10 Photo Gallery

From this project, I learned more about the inner workings of WordPress as well as the PHP functions that are built into the platform.
